# HXProf

Profiling integrated in PHP

- **Use-case:** Get performance report
- **When to start:** as soon as performance matters
- **When to run:** With every commit/pull-request
- **Provides:** callgraph, html report

Last commit was 26 Feb 2015

# Blackfire

Best PHP profiling available. (I'm paid to say this)

- **Use-case:** Get detailed performance report
- **When to start:** as soon as performance matters
- **When to run:** With every commit/pull-request
- **Provides:** detailed report on blackfire.io

Provides a PHPUnit integration for assertions based on profile.
